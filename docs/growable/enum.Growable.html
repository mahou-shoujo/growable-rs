A chunk of heap memory that can be assigned to a struct or a trait object. Until assigned to some data it behaves similarly to a Box<[u8; N]>, it can be cloned and would be dropped if leaves the scope.

Examples

First, let's spawn a new Growable. In this case no allocation would be performed on init.

  let growable = Growable::new();

Now we can assign some data to it.

  let arr: Reusable<[char; 3]> = growable.assign(['f', 'o', 'o']);
  assert_eq!(&*arr, &['f', 'o', 'o']);

Unwanted data could be then freed on demand, fetching Growable back. Then it could be assigned to some data again and so on.

  let growable = arr.free();
  let arr: Reusable<[char; 6]> = growable.assign(['f', 'o', 'o', 'b', 'a', 'r']);
  assert_eq!(&*arr, &['f', 'o', 'o', 'b', 'a', 'r']);

Returns allocated on the heap struct, an actual (re)allocation will be performed only if there is not enough space or the pointer alignment is invalid. Additionally stores a pointer to the vtable creating trait object.

Examples

  trait Answerer {

      fn get_answer(&self) -> u32;
  }

  struct Foo;

  impl Answerer for Foo {

      fn get_answer(&self) -> u32 { 42 }
  }

  let growable = Growable::new();
  let foo: Reusable<Answerer> = growable.assign_as_trait(Foo);
  assert_eq!(foo.get_answer(), 42);
